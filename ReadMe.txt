Welcome to Toy Robot
Commands available:
1. The place command allows the player to initialize the robot's location and direction on a board.

        place x,y,direction
        ex: place 2,2,east

        where x and y are coordinates
        and direction refers to north, south, east, west

2. The move command instructs the robot to move 1 square in the direction it is facing.
3. The left command instructs the robot to rotate 90░ counterclockwise.
4. The right command instructs the robot to rotate 90░ clockwise.
5. The report command outputs the robot's current location on the tabletop and the direction it is facing.
6. The clear command to clear the screen.
7. The exit command exits the game.
